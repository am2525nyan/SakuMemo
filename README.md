# SakuMemo

AIを活用したタスク・メモ管理アプリ（iOS）

## どんなアプリか

SakuMemoは、メモを入力するだけでAIが自動的に**カテゴリ分類**と**優先度判定**を行うスマートメモアプリです。
日常のタスクや買い物リスト、やりたいことなどを素早く記録でき、優先度は時間経過とともに自動的に低下・アーカイブされるため、本当に大切なことに集中できます。
どんどん新しい技術を導入して使って行っています。
### 対象プラットフォーム

- iOS 18.4+

## 実装した内容

### コア機能

- **メモの作成・管理** - フローティングボタンからサクッとメモを追加。スワイプで削除・アーカイブ
- **AI自動分析**
  - **Google Gemini API** - Firebase AI経由で重要度（0.0〜1.0）とカテゴリ（todo / 買い物 / やりたいこと）を自動判定
  - **Apple Intelligence** (iOS 26+) - オンデバイスAIによるプライバシー重視の分析。利用不可時はGeminiにフォールバック
- **タスク自動抽出** - 複雑な文章から個別のタスクを抽出して一括登録（例：「友達が家にくるので片付ける」→ 掃除機をかける、布団を洗濯する、お菓子を買うetc...)

### 優先度システム

- AIが初期優先度を自動設定（0.0〜1.0）
- 設定可能な日数経過後に優先度が自動低下
- 絵文字で視覚的に表示（🔥 高 / 🌤 中 / ❄️ 低）
- 優先度がゼロになったメモは自動アーカイブ

### 期日・通知

- メモに期限を設定可能
- AIが生成するフレンドリーなリマインドメッセージ
- 通知にAI生成画像を添付（iOS 18.4+）

### その他の機能

- **アーカイブ** - 完了・期限切れメモの管理と復元
- **検索・フィルタリング** - テキスト検索、カテゴリ・優先度でのフィルタ
- **サブスクリプション** - フリーミアムモデル（無料：1日3件 / プレミアム：無制限）
- **Siriショートカット** - 音声でメモを追加
- **広告** - Google AdMobバナー広告

### タブ構成

| タブ | 内容 |
|------|------|
| ホーム | メモ一覧・追加・期日間近のメモ表示 |
| アーカイブ | アーカイブ済みメモの管理 |
| 設定 | 自動アーカイブ・優先度低下の設定 |

## アーキテクチャ・技術スタック

### アーキテクチャ

- **TCA (The Composable Architecture)** - 単方向データフローによる状態管理
- **SPM マルチモジュール構成** - 機能ごとにモジュール分割

### モジュール構成

```
SakuMemoPackage/Sources/
├── AppFeature/           # タブベースのアプリコンテナ
├── MemoFeature/          # ホームタブ（メモ一覧）
├── ArchiveFeature/       # アーカイブタブ
├── SettingsFeature/      # 設定タブ
├── AddMemoFeature/       # メモ作成・タスク抽出
├── MemoDetailFeature/    # メモ編集・通知設定
├── SubscriptionFeature/  # サブスクリプション管理
├── SearchFeature/        # 検索・フィルタ
├── AppIntent/            # Siriショートカット
├── Repository/           # データ・API層
├── RepositoryProtocol/   # リポジトリインターフェース
├── SharedModel/          # 共有データモデル
├── Components/           # 再利用可能なUIコンポーネント
└── Utils/                # ユーティリティ
```

### 主要な依存ライブラリ

| パッケージ | バージョン | 用途 |
|-----------|-----------|------|
| swift-composable-architecture | 1.19.1 | 状態管理 |
| Alamofire | 5.10.2 | HTTP通信 |
| PopupView | 4.1.7 | トースト通知 |
| firebase-ios-sdk | 12.0.0 | Gemini AI / App Check |

### システムフレームワーク

- **UserNotifications** - 通知

- **FoundationModels** (iOS 26+) - Apple Intelligence
